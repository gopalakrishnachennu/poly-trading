# ADR 0057: Canary Code Eligibility Grants No Capital Authority

- Status: Accepted
- Date: 2026-07-22

## Decision

Phase 3.7 implements and certifies canary controls without wallet access,
capital allocation or execution authority. A locally eligible report proves
only that deterministic cases enforced exact complete-set scope, allowlists,
tiny ceilings, dual control and emergency responses.

Real canary activation requires separate current environment evidence, legal
eligibility, funded-wallet authority and explicit operators outside this
repository. `NO_TRADE` is always permitted and is a successful safe result.
